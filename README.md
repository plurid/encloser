<h1 align="center">
    encloser
</h1>


<h3 align="center">
    Imagene Builder
</h3>


`encloser` is a service or self-hosted system for building imagenes.

`encloser` uses [`plurid`](https://github.com/plurid/plurid) to explore information as a 3D structure.
